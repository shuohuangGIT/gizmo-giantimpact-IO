I share scripts for generating GIZMO-readable hdf5 files and ploting

1. makeplanet_gizmo.py:
use woma to make single planet and save the planet particles into GIZMO-readable hdf5 file. Initialize planet mass and composition

2. combineplanet_gizmo.py:
combine two planets in two different hdf5 files, initilize impact separation, velocity, angle

3. makeplanet_plot.ipynb:
plot hdf5
